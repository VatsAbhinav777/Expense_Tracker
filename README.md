# Expense_Tracker

# Overview of the Project
The Expense Tracker is a lightweight, console-based application designed to help users
manage their daily financial outgoings. In a world where small expenses often go unnoticed, this tool
provides a quick and easy way to log transactions and visualize total spending in real-time. It operates
entirely in the terminal, ensuring zero distractions and instant execution.

# Features
1) Add Expenses: Quickly input an amount and a brief description for any transaction.
2) View Summary: Display a clean, tabulated list of all expenses recorded in the current session.
3) Automatic Totals: Instantly calculates and displays the grand total of your spending.
4) Interactive Menu: Simple navigation system (Add, View, Exit) suitable for all users.
5) Error Handling: Basic validation ensures the application runs smoothly.

# Technologies used/Tools
1) Language: Python 3
2) Interface: Command Line Interface (CLI) / Terminal
3) Libraries: Standard Python libraries (no external dependencies required)

# Steps to install & run the project
1. Prerequisites: Ensure you have Python installed on your system. You can verify this by typing
python --version in your terminal.
2. Download: Clone this repository or download the expense_tracker.py file.
git clone [https://github.com/your-username/expense-tracker.git](https://github.com
3. Navigate: Open your terminal or command prompt and navigate to the folder containing the file.
cd expense-tracker
4. Run: Execute the script using the python command:
python expense_tracker.py

# Instructions for testing
1. Start the App: Run the script. You should see the main menu.
2. Test Adding: Select option 1 . Enter an amount (e.g., 50 ) and a description (e.g., Snacks ).
3. Test Viewing: Select option 2 . Verify that "Snacks" appears with the correct price.
4. Test Calculation: Add a second expense (e.g., 100 ). Select option 2 again and verify the total
matches the sum (e.g., 150.0 ).
5. Test Exit: Select option 3 to close the application.
